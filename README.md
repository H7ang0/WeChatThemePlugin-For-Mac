# WeChatThemePlugin-For-Mac
Mac微信主题插件

本项目是 **微信主题插件下载库**，提供丰富的微信主题供用户选择和下载使用。请注意：此项目 **不是开源项目**，仅作为主题下载库使用。

---

## 📋 功能介绍

- **主题下载**：微信主题，一键下载快速应用。  
- **适配版本**：支持最新微信版本，确保主题兼容性。  
- **社区互动**：加入 Telegram 群组，与其他用户交流主题使用经验。

---

## 📥 使用说明

1. 下载并安装微信主题插件。  
2. 打开插件后，进入主题商店界面。  
3. 浏览主题列表，选择喜欢的主题并下载。  
4. 应用主题后，重启微信即可生效。
